HtML is Hypertext Markup Language

What is HTML

Hypertext markup Language is the set of markup symbols or code inserted into a file intended to display on the internet. The markup tells the browser how to display a web's word, photo, video and other content. Html first introduced by Tim Berners-Lee and it is a standard markup language for creating web page.

CSS Stand for Cascading Style Sheets

What is CSS

CSS specifies your document's style-page layouts, colors and fonts are all determined with css. Using CSS, can control text color, font size, changing background and more... In a simple word css is a language that is responsible for styling a web page.
